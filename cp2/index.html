E-Commerce Customer Churn Predictor

Upload a preprocessed customer feature CSV and run a deep learning model directly in your browser using TensorFlow.js.

1. Upload Customer Data (CSV)

Upload a CSV file containing multiple rows of preprocessed numeric features in the same column order as used during model training in your Colab notebook.
Example: the full X_test_scaled DataFrame saved with .to_csv("customers_scaled.csv").

CSV requirements:
  • Header row with feature names.
  • All values must be numeric (already encoded & scaled).
  • Each data row = one customer.
  • Number of columns must match the model input size.
